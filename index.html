  1. Implementar un algoritmo que reciba 2 argumentos y los sume, el resultado se deberá imprimir en pantalla

    Inicio
    Variables: a, b, resultado
    Escribir ("Introducir primer número")
    Leer (a)
    Escribir ("Introducir segundo número")
    Leer (b)
    resultado = a + b
    Escribir ("La suma es: ", resultado)
    Fin

  2. Un estudiante realiza 4 exámenes, calcular el promedio de estos

    Inicio
    Variables: nota1, nota2, nota3, nota4, notaFinal
    Escribir ("Introducir Nota 01")
    Leer (nota1)
    Escribir ("Introducir Nota 02")
    Leer (nota2)
    Escribir ("Introducir Nota 03")
    Leer (nota3)
    Escribir ("Introducir Nota 04")
    Leer (nota4)
    notaFinal =(nota1 + nota2 + nota3 + nota4) / 4
    Escribir ("El promedio es: ", notaFinal)
    Fin

  3. Calcular el área de un rectángulo

    Inicio
    Variables: altura, base, area
    Escribir ("Introducir altura")
    Leer (altura)
    Escribir ("Introducir base")
    Leer (base)
    area = base * altura
    Escribir ("El área del rectangulo es: ", area)
    Fin

  4. Calcular el área de un triángulo

    Inicio
    Variables: altura, base, area
    Escribir ("Introducir altura")
    Leer (altura)
    Escribir ("Introducir base")
    Leer (base)
    area = base * altura / 2
    Escribir ("El área del triangulo es: ", area)
    Fin

  5. Calcular el área de una circunferencia

    Inicio
    Variables: radio, area, pi=3.1415
    Escribir ("Introducir radio")
    Leer (radio)
    area = pi * (radio^2)
    Escribir ("El área de la circunferencia es: ", area)
    Fin

  6. Determinar el sueldo semanal de un trabajador basándose en sus horas trabajadas y su salario de hora hombre

    Inicio
    Variables: hora_t, pago_xh, sueldo_sem
    Escribir ("Introducir Horas trabajados")
    Leer (hora_t)
    Escribir ("Introducir pago por hora")
    Leer (pago_xh)
    sueldo_sem = hora_t * pago_xh
    Escribir ("El pago semanal es: ", sueldo_sem)
    Fin

  7. Una modista, para realizar sus prendas de vestir, encarga las telas al extranjero. Para cada pedido, tiene que proporcionar las medidas de la tela en pulgadas, pero ella generalmente las tiene en metros. Realice un algoritmo para ayudar a resolver el problema, determinando cuantas pulgadas debe pedir con base en los metros que requiere. Represéntelo mediante el diagrama de flujo y el pseudocódigo (1 pulgada = 0.0254 m).

    Inicio
    Variables: metros, pulgada=0.0254, resultado
    Escribir ("Introducir Metros")
    Leer (metros)
    resultado = metros * pulgada
    Escribir ("Pedir: ", resultado , "pulgadas")
    Fin
